# Unsplash API를 활용한 이미지 검색 사이트

### 개발환경
 - React, Typescript, Next.js, Scss
 - vercel (배포)
 - 라이브러리: material-ui, axios

### 실행방법
  ```
  1. git clone
  2. cd photo-web-apllication
  3. yarn  (패키지 설치)
  4. yarn dev (실행)
  ```
